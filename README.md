# OwnObjectScriptExtension

A VSCode Extension for ObjectScript users.
Adds <code>Set</code>, <code>Do</code> and <code>Write</code> modifier to your ObjectScript code.

## Features

Add modifiers with the <code>Add ObjectScript Modifier</code> Command. After successfully adding the modifiers, you will receive a message with the lines where Modifier where added.

<img alt="Gif" src="https://github.com/phil1436/ownobjectscriptextension/raw/master/resources/demo.gif" width="100%"/>

> Tip: You can use IntelliSense even if you have errors in your code.

## Requirements

The <a href= "https://intersystems-community.github.io/vscode-objectscript/">ObjectScript Extension</a> should be installed. And an active texteditor with an ObjectScript file should be open.

## Install

Clone this repository under <code>~/.vscode/extensions</code>.

<pre>
git clone https://github.com/phil1436/ownobjectscriptextension C:\Users\your-user\.vscode\extensions\ownobjectscriptextension
</pre>
